# PlanningScrumPorker

# インストール
```
git clone https://github.com/35d/PlanningScrumPorker.git
yarn install
yarn start
```

# TypeScript
```
yarn tsc // => typescript compile ./src/*.ts or ./src/*.tsx
yarn tsc -w // watch mode
```
